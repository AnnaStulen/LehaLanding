## Как вписать risk-картинки в дизайн

Risk-картинки сильно отличаются по стилю, поэтому нельзя вставлять их сырыми как обычные картинки.

Правильная интеграция:

- оставить саму картинку читаемой;
- не делать глобальное затемнение;
- не ставить `opacity` ниже 1 на изображение;
- не использовать `brightness()`, `hue-rotate()` или тяжелые global filters;
- положить картинку в square technical frame;
- вокруг сделать черный/deep-blue stage;
- добавить cyan/blue edge glow;
- добавить тонкие technical borders;
- добавить local labels на маленьких темных подложках;
- добавить moving light sweep / scanline / gloss highlight поверх frame layer, а не поверх всей картинки как затемнение;
- добавить hover zoom/depth/glow;
- если бежевый фон слишком выбивается, не перекрашивать весь скрин, а сделать вокруг него темный технический контекст и маленькие cyan callouts.
